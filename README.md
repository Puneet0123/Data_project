**Goal:** Predict whether a loan application will be approved based on financial and demographic data.

- Visualized feature importances and model evaluation metrics.

### 2. Customer Segmentation (E-Commerce)

**Goal:**  
To identify distinct customer segments based on purchasing behavior for targeted marketing using RFM analysis and clustering.

**Key Highlights:**
- Performed in-depth EDA and feature engineering on UK-based Online Retail dataset.
- Applied **RFM analysis** and used **KMeans clustering** to segment customers into actionable groups.
- Developed interactive visual dashboards for business insights.
- Delivered marketing strategy suggestions for each segment.

**Tools & Techniques:**  
Pandas, Matplotlib, Seaborn, KMeans, Scikit-Learn, Plotly, Streamlit

[Project Folder](https://github.com/Puneet0123/Data_project/tree/main/customer_segmentation)
